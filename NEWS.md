## Version 0.2.0 (2026-07-21)

### Breaking changes

- Processing of the user-generated YAML file is now done in 2 stages: (1) the YAML file is
  completed according to the calibration data found in directory and (2) the completed YAML
  file is processed to perform the calibration.

- Scanner removed.
